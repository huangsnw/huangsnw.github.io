博客分发方案

域名都买很久了，但是一只因为江苏的备案政策原因没有备案。最近想到个绕过江苏政策的办法，那就是在户籍地备案，结果等了一两周，顺利通过！

那我现在就可以将我的博客放在自己的服务器上了。

我使用方案是：typora + picGo + github图床、仓库 + github webhook

Typora可以说是最好的markdown编辑器了吧。虽然Typora现在收费了，但是在某些地方仍然可以找到历史版本。

然后在github上创建一个图床仓库，创建token，使用picGo作为图片上传工具。

配置妥当后在github上再开一个github pages仓库，配置hexo配置文件，使用hexo g && hexo d同步文件。

下面是服务器的部分：

webhook是go语言编写的，所以在自己的linux服务器上安装golang环境。

yum install go

安装webhook工具。

go install github.com/adnanh/webhook@latest

国内服务器可能拉取github的代码，可能会出现超时。但是我发现使用ssh比https的方式稳定一点。

安装git。

yum install git

编写shell脚本，给权限。

写hooks.json文件

  • id 顾名思义, 可以自定义
  • execute-command 执行脚本名称
  • command-working-directory 脚本所在目录

启动webhook

在github页面新建webhook。

OK！

webhook成功执行。
